隐私政策

一、隐私声明

本博客（以下简称”我们”）深知个人隐私的重要性，严格遵守《中华人民共和国个人信息保护法》等相关法律法规，致力于保护每一位访客的个人信息安全与隐私权利。

我们承诺：

  • 仅在合法、正当和必要的范围内收集、使用和存储个人信息
  • 采取严格的安全措施保护您的个人信息
  • 不会将您的信息用于与本政策无关的目的
  • 尊重并保障您对个人信息的各项权利

二、信息收集

我们在您访问、浏览或使用本博客服务时，可能会收集以下类型的信息：

2.1 自动收集的信息

  • 网络身份标识信息

    • IP 地址
    • 浏览器用户代理字符串（UA）
    • Cookie 与本地存储标识

  • 设备与环境信息

    • 操作系统类型与版本
    • 浏览器类型与版本
    • 时区设置
    • 性能与网络加载情况

  • 访问日志

    • 访问时间、访问页面、停留时长
    • 来源网站与点击行为

2.2 您主动提供的信息

当您在本博客进行评论或其他互动时，可能会主动提供以下信息：

  • 昵称
  • 电子邮箱
  • 个人网站地址
  • 评论内容

三、第三方服务

为了提升博客性能、分析访客行为及美化界面，我们会使用以下第三方服务：

3.1 分析与统计服务

  • 百度统计 - 提供网站流量分析服务
  • Google Analytics - 提供匿名流量统计与用户行为分析
  • Microsoft Clarity - 提供热图与用户行为录制分析
  • 51a 统计平台 - 提供页面访问计数与来源分析

3.2 性能与安全服务

  • Cloudflare CDN与安全服务 - 提供内容分发、DDoS防护与安全加固

3.3 资源服务

  • 阿里巴巴 Iconfont 图标库 - 提供网站图标资源

这些第三方服务可能会根据其自身的隐私政策收集和处理您的信息。我们建议您访问各服务提供商的隐私政策页面，了解他们的具体做法。

四、Cookie 政策

4.1 什么是 Cookie

Cookie 是您访问网站时放置在您的浏览器或设备中的小型文本文件，常包含标识符、站点名称以及一些号码和字符。Cookie 不会直接识别您的身份，但可帮助我们：

  • 记录您的偏好（例如深色模式、语言设置）；
  • 保障网站安全与基本功能；
  • 统计网站流量并分析性能，以便改进内容和用户体验；
  • 提供个性化内容推荐。

4.2 我们使用的 Cookie 类型

类别 名称 提供方 用途 保留期限
必要 Cookie hexo_theme_mode 本站 记录深色/浅色主题偏好 1 年
统计 Cookie _clck, _clsk Microsoft Clarity 匿名会话热图与漏斗分析 1 年
统计 Cookie _ga, _ga_* Google Analytics 匿名流量统计与用户行为分析 2 年
统计 Cookie _51a 51a 统计 页面访问计数与来源分析 1 年
统计 Cookie __cfduid, __cflb Cloudflare Analytics 站点性能与访问日志 1 个月

重要说明：第三方 Cookie 仅在您明确同意后才会被写入。拒绝此类 Cookie 不会影响博客的基础功能。

4.3 如何管理或删除 Cookie

您可以通过以下方式管理或删除 Cookie：

  • 浏览器设置：多数浏览器「设置 → 隐私与安全 → Cookie」中可查看、阻止或删除 Cookie。
  • 无痕/隐私模式：使用浏览器无痕窗口可默认阻止持久化 Cookie。
  • Cookie 同意横幅：本博客提供的 Cookie 同意横幅中可以选择接受或拒绝非必要的 Cookie。

4.4 数据跨境传输

我们选用的 Microsoft Clarity 与 Google Analytics 等服务的服务器可能位于中国境外。对于这些跨境传输的数据，我们确保：

  • 相关处理遵循欧盟标准合同条款（SCCs）、**英国国际数据传输协议（IDTA）与加州消费者隐私法案（CCPA）**相关跨境传输要求
  • 采取适当措施确保您的数据获得与中国境内同等水平的保护
  • 仅传输提供服务所必需的信息

五、评论系统

我们使用无登录系统的匿名评论功能，您可以自愿选择填写相关信息。

5.1 评论时收集的信息

在您发表评论时，我们可能会收集以下信息：

  1. 昵称：用于在评论区显示您的身份（主动提供，公开显示）
  2. 电子邮箱：用于发送评论回复通知（主动提供，不会公开显示）
    • 如填写 QQ 邮箱或 QQ 号，系统可能使用该信息获取您的 QQ 头像
    • 如不想展示 QQ 头像，可填写不含 QQ 号的 QQ 邮箱
  3. 个人网站地址：用于点击头像时快速访问您的网站（主动提供，公开显示）
  4. IP 地址：作为反垃圾评论的判别依据（被动收集，不会公开 IP，仅显示 IP 所在城市）
  5. 浏览器用户代理：用于识别设备信息，帮助我们改进用户体验（被动收集，部分信息可能公开显示）

5.2 评论信息的使用

我们使用评论信息的目的包括：

  • 展示您的评论内容
  • 回复您的问题或反馈
  • 防止垃圾评论和恶意行为
  • 改进网站功能和用户体验

六、信息使用

我们收集的个人信息主要用于以下目的：

  1. 提供和改进服务：确保博客正常运行，分析与优化网站性能，提高内容质量与用户体验
  2. 统计分析：针对访问趋势与用户行为进行统计与分析，以便优化内容和服务
  3. 安全保障：防范垃圾评论、恶意访问和网络攻击，保障博客及用户访问的安全性
  4. 个性化服务：根据您的偏好提供个性化内容和设置（如主题模式）
  5. 通信交流：当您通过评论留下邮箱时，用于发送评论回复通知

未经您明确同意，我们不会将您的个人信息用于上述目的之外的任何用途。

七、信息共享与披露

7.1 一般情况

我们不会向无关第三方出售、出租或共享您的个人信息。

7.2 例外情况

在下列情况下，我们可能会披露您的个人信息：

  1. 获得您的明确同意时
  2. 法律法规、诉讼或监管机关要求时
  3. 为维护博客安全、防范欺诈或恶意行为所必需时
  4. 与我们的服务提供商共享，但仅限于提供服务所必需的信息，并要求其遵守严格的保密义务

八、信息安全

我们高度重视您的个人信息安全，采取了一系列合理可行的安全措施，包括但不限于：

  1. 技术措施：采用数据加密、安全套接层(SSL)、访问权限控制等技术保护数据安全
  2. 管理措施：实施严格的内部安全管理制度，定期进行安全审核和员工培训
  3. 传输安全：确保信息在传输过程中受到保护，防止数据泄露或被篡改
  4. 存储安全：对存储的个人信息进行分类管理，并设置访问权限

尽管我们已采取合理措施保护您的信息，但互联网并非绝对安全的环境，我们无法保证信息在互联网传输过程中的绝对安全。

九、用户权利

根据相关法律法规，您有权随时行使以下权利：

  1. 访问权：了解我们所持有的关于您的个人信息
  2. 更正权：更新不准确或不完整的个人信息
  3. 删除权：在特定情况下，请求删除您的个人信息
  4. 限制处理权：在法律允许范围内，请求限制对您个人信息的处理
  5. 反对权：基于特定理由，反对我们对您个人信息的处理
  6. 撤回同意权：对基于您同意的个人信息处理行为提出撤回
  7. 数据可携带权：在特定情况下，以结构化、常用和机器可读的格式接收您提供的个人信息

9.1 如何行使权利

如需行使上述权利，请发送邮件至下方”联系我们”中提供的邮箱，我们将在收到请求后的15个工作日内回复您的请求。

9.2 例外情况

在以下情况下，我们可能无法满足您的请求：

  • 与我们履行法律法规规定的义务相关的
  • 与国家安全、国防安全直接相关的
  • 与公共安全、公共卫生、重大公共利益直接相关的
  • 与刑事侦查、起诉、审判和执行判决等直接相关的

十、数据保留

10.1 保留期限

我们仅在实现信息收集目的所必需的期限内保留您的个人信息：

  • 评论数据：保留至评论被删除或博客停止运营
  • 访问日志：一般保留90天，之后进行匿名化处理
  • Cookie数据：根据不同类型的Cookie，保留期限从1个月到2年不等（详见Cookie表格）
  • 其他信息：在实现收集目的所需的最短时间内保留，除非法律要求更长的保留期限

10.2 数据处理

达成收集目的后，我们会在法律规定或业务需求允许的范围内：

  • 删除不再需要的个人信息
  • 对需要继续保存的数据进行匿名化或去标识化处理
  • 确保数据删除或匿名化处理后无法识别特定个人

十一、政策更新

11.1 更新原因

我们可能会根据以下情况更新本隐私政策：

  • 法律法规的变化
  • 业务功能的变更
  • 个人信息处理方式的调整
  • 安全措施的更新

11.2 更新通知

当我们对隐私政策进行重大变更时，我们将：

  • 在页面显著位置发布最新版本并标注更新日期
  • 可能通过适当方式（如网站公告）通知您

11.3 您的选择

继续使用本博客服务，即表示您同意修订后的隐私政策。如您不同意更新后的隐私政策，您应停止使用本博客服务。

我们建议您定期查阅本隐私政策，以便了解最新内容。

十二、安全防护措施

12.1 黑名单机制

为保护博客安全和用户体验，当监测到存在恶意访问、恶意请求、恶意攻击、恶意评论等行为时，我们可能会：

  • 将相关IP地址、设备信息等添加到临时黑名单
  • 对黑名单中的用户采取限制访问措施
  • 保存相关证据用于安全分析和可能的法律程序

12.2 信息共享

在必要情况下，为了保护公共利益和其他用户的合法权益，我们可能会：

  • 与相关安全机构共享黑名单信息
  • 向其他受影响的站点提供必要的安全信息（如IP地址、攻击特征等）

这些措施仅针对实际发生的恶意行为，我们会确保在法律框架内合理使用这些信息。

十三、联系我们

如对本隐私政策有疑问、建议或需行使相关权利，请通过以下方式与我们联系：

  • 邮箱：wybtgrw@outlook.com
  • 博客留言区：在本博客相关页面提交您的反馈

我们将在收到您的请求后的15个工作日内回复您。

本隐私政策自发布之日起生效。
感谢您对本博客的信任与支持！
